# 💰 AI Smart Budget Agent

## Overview

AI Smart Budget Agent is an AI-powered financial assistant that analyzes PhonePe transaction statements and generates personalized budgeting recommendations.

The application extracts transactions from PDF statements, categorizes spending, analyzes expenses, visualizes spending patterns, and generates AI-based financial advice using Groq LLM.

## Technologies Used

* Python
* Streamlit
* Pandas
* Groq API
* PDFPlumber
* Matplotlib

* Spending Analysis
* Budget Planning
* AI Financial Suggestions
* Data Visualization

## Installation

streamlit run app.py

## Workflow

PhonePe PDF → Transaction Parser → Expense Analysis → Category Detection → Budget Planning → Groq AI Suggestions
